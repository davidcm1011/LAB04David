¿Cómo elimino una rama creada en git?
    Para eliminar una rama en Git, puedes utilizar el comando git branch -d <nombre-de-la-rama> 
    si deseas eliminar una rama que ya ha sido fusionada en la rama actual.
    Si deseas eliminar una rama sin importar si ha sido fusionada o no, puedes usar git branch -D <nombre-de-la-rama>.
¿Por qué debería de usar la ramificación en proyectos?
    se dberia usar la ramificacion por:
    -desarrollo aislado
    -facilidad de colaboracion
    -experimentacion segura
    -gestion de versiónes
    -facilita la revision de codigo.
